## 🔑 핵심 기능
- 회원가입 및 세션 기반 로그인
- 게시물 CRUD 및 댓글 기능
- 게시물 비공개 설정
- 내 정보 수정 및 비밀번호 찾기

----

## 🔍 담당 업무
- 담당 페이지 UI 및 기능 구현
- 모듈화된 API 통신 시스템 구축
- Redux Persist를 활용한 상태 영속성 관리
- Styled-components를 활용한 컴포넌트 단위 스타일링 구현
- Private router를 통한 인증 상태에 따른 라우팅 접근 제어
- 정규식을 활용한 유효성 검사
- Vercel 배포 및 AWS Route 53 도메인 관리
- S3 Presigned URL 이미지 업로드
- 이메일 인증 프로세스 구현
- Git 브랜치 관리 및 PR을 이용한 협업 시스템 구축
